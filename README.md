📌 Black Glass Calculator – Project Description

The Black Glass Calculator is a modern, responsive web application built using HTML, CSS, JavaScript, and Bootstrap. Designed with a sleek black glassmorphism UI, the calculator combines functionality with a premium, futuristic aesthetic, featuring a blurred glass background, glossy buttons, and smooth hover and click animations.

The application provides basic arithmetic operations including addition, subtraction, multiplication, division, and decimal calculations. Users can input values using the on-screen buttons and view the results in a clean, intuitive display. The interface includes a dedicated clear button for resetting calculations.

With increased width and spacious buttons, the calculator is optimized for both desktop and tablet screens. The design incorporates orange accent colors and dynamic click animations to enhance interactivity and user experience.

This project demonstrates proficiency in frontend web development, including responsive design, CSS effects, interactive JavaScript functionality, and modern UI trends.

🚀 Key Features

Modern black glassmorphism UI with glossy shine

Fully responsive design (desktop & tablet friendly)

Basic arithmetic operations (+, -, *, /, decimal)

Dynamic hover and click animations

Clear button for resetting calculations

Built using HTML, CSS, JavaScript, and Bootstrap
